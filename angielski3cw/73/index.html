Uzasadnienie: konstrukcja I wish + past perfect, by wyrazić żal odnośnie sytuacji z przeszłości. III tryb warunkowy, by mówić o przeszłości: if + past perfect, would have + past participle (III kolumna w tabeli czasowników nieregularnych lub końcówka -ed).

2. tried, would be

Uzasadnienie: II tryb warunkowy, by mówić o hipotetycznej sytuacji w teraźniejszości lub przyszłości: if + past simple, would + bezokolicznik.

3. were, would send, hadn't been, would have joined

Uzasadnienie: (X) II tryb warunkowy, by mówić o hipotetycznej sytuacji w teraźniejszości lub przyszłości: if + past simple, would + bezokolicznik. (Y) III tryb warunkowy, by mówić o przeszłości: if + past perfect, would have + past participle.

4. would you do, were, would get

Uzasadnienie: (X) II tryb warunkowy, by mówić o hipotetycznej sytuacji w teraźniejszości lub przyszłości: if + past simple, would + bezokolicznik. Zadanie 5

1. had  

Uzasadnienie: III tryb warunkowy, by mówić o przeszłości: if + past perfect (had + past participle), would have + past participle.

2. been

Uzasadnienie: be happy - cieszyć się. Konstrukcja w III trybie warunkowym: would(n't) have + past participle, if + past perfect.

3. stole

Uzasadnienie: połączenie czasu past continuous (was/were + czasownik z -ing) i czasu past simple (końcówka -ed lub II kolumna w tabeli). Konstrukcja: while + past continuous, past simple.

4. only

Uzasadnienie: konstrukcja if only + past perfect, by wyrazić żal odnośnie sytuacji w przeszłości.

5. service

Uzasadnienie: do community service - wykonywać prace społeczne (rodzaj kary).


Zadanie 6

1. if you had a

Uzasadnienie: II tryb warunkowy, by opisać nierealną, hipotetyczną sytuację w przyszłości: would + bezokolicznik, if + past simple. Używamy przedimka nieokreślonego przed rzeczownikiem po luce, ponieważ nie mówimy o czymś konkretnym i wspominamy o tym po raz pierwszy.

2. I would have played in

Uzasadnienie: III tryb warunkowy, by mówić o przeszłości: if + past perfect, would have + past participle (III kolumna w tabeli czasowników nieregularnych lub końcówka -ed).

3. If only we had noticed 

Uzasadnienie: konstrukcja if only + past perfect (had + past participle), by wyrazić żal odnośnie konkretnej sytuacji z przeszłości.

4. had been the judge

Uzasadnienie: III tryb warunkowy, by mówić o przeszłości: if + past perfect (had + past participle), would have + past participle. Używamy przedimka określonego przed rzeczownikiem judge (sędzia), ponieważ to zdanie dotyczy konkretnej sytuacji ogłoszenia wyroku, w której uczestniczył konkretny/a sędzia/sędzina.
