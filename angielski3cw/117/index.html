Zadanie 3

Uwaga!

Drugi okres warunkowy to konstrukcja, która służy do mówienia o nierealnych, hipotetycznych sytuacjach w teraźniejszości lub przyszłości.

Tworzymy go według następującej konstrukcji:

if + past simple, would + bezokolicznik
(warunek) (rezultat)


Konstrukcji I wish/If only używamy z czasem past simple, jeśli chcemy wyrazić życzenie, żeby obecny albo przyszły stan rzeczy był inny. Używamy też tej formy, żeby wyrazić żal odnośnie własnej osoby.

W obu konstrukcjach możemy użyć formy were zamiast was w pierwszej i trzeciej osobie liczby pojedynczej, ponieważ jest to tryb unreal past (przeszłości nierzeczywistej).

 

1. 

I wish I had a job. If I had a job, I would earn some money.

If I had some money, I would buy myself some new clothes.


2.

If only I was/were seventeen. If I was/were seventeen, I would be able to drive.

If I was/were able to drive, I wouldn't have to travel by bus.
