Zadanie 3

1. should come  should have come

Uzasadnienie: wyrażenie żalu, że coś się nie wydarzyło w przeszłości - should have + past participle.

2. can could

Uzasadnienie: wyrażenie żalu z powodu obecnej sytuacji - wish + past simple.

3. set off had set off

Uzasadnienie: wyrażenie niezadowolenia z sytuacji z przeszłości - If only+ past perfect.

4. clean cleaned

Uzasadnienie: wyrażenie życzenia dotyczącego teraźniejszości lub bliskiej przyszłości - It's time + podmiot + past simple.

5. don't have to mustn't 

Uzasadnienie: zakaz - mustn't + bezokolicznik.

6. doesn't cook didn't cook 

Uzasadnienie: preferencje i życzenia dotyczące teraźniejszości i przyszłości - would rather + podmiot + past simple.

7. ought see ought to see

Uzasadnienie: wyrażenie rady - ought to + bezokolicznik. 

8. must fall asleep must have fallen asleep

Uzasadnienie: wyrażenie pewności do zdarzenia z przeszłości - must have + past participle.
