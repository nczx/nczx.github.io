Zadanie 1

Żeby opisać hipotetyczną sytuację z przeszłości i jej skutki, które widzimy w teraźniejszości używamy mieszanego trybu warunkowego w następującej formie:

 if + past perfect, would + bezokolicznik

 

Żeby opisać hipotetyczną sytuację w teraźniejszości i jej wpływ na przeszłość używamy mieszanego trybu warunkowego w następującej formie:

 if + past simple, would + have + past participle.

 

Możliwe poprawne zdania:

1. If I hadn't had any homework, I would practise the guitar.

If I didn't have any homework, I would have practised the guitar. 

2. If we hadn't had to wait for Toby to arrive yesterday, we wouldn't miss the train.

3. If he wasn't so moody, he would have had more friends. 

If he hadn't been so moody, he would have more friends.

4. She would be very happy if she hadn't been chosen to play in the match last Saturday. 

5. I wouldn't answer my phone if the call had been from an unknown number. 

6. If you could have changed one thing about yourself, what would it be?

Zadanie 4

1. long

Uzasadnienie: sytuacja, która może wydarzyć się w przyszłości - first conditional; as long as (o ile) zamiast if (jeśli).

2. have

Uzasadnienie: sytuacja oczywista, która ma łatwe do przewidzenia następstwa - zero conditional.

3. provided

Uzasadnienie: zasada, reguła - zero conditional; provided that (pod warunkiem, że) zamiast if (jeśli). 

4. had 

Uzasadnienie: hipotetyczna sytuacja z przeszłości i jej niedoszłe przeszłe następstwa - third conditional.

5. would

Uzasadnienie: hipotetyczna sytuacja z przeszłości i jej skutki w teraźniejszości - if + past perfect, would + bezokolicznik.

6. unless

Uzasadnienie: sytuacja, która może wydarzyć się w przyszłości - first conditional; unless (jeśli nie) zamiast if (jeśli).
