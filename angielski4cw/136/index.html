Zadanie 1

1. when

Uzasadnienie: w zdaniach opisujących sytuację z przeszłości przerwaną przez inne zdarzenie zwykle stosujemy when - (kiedy).

2. rush

Uzasadnienie: przypływ adrenaliny.

3. run

Uzasadnienie: run into - wpaść na, napotkać.

4. as

Uzasadnienie: as fast as possible - tak szybko, jak to możliwe.

5. had

Uzasadnienie: sytuacja, która miała miejsce przed innym zdarzeniem z przeszłości - czas past perfect. 


Zadanie 2

1. used

Uzasadnienie: stan z przeszłości - used to + bezokolicznik.

2. been 

 Uzasadnienie: sytuacja, która miała miejsce przed innym zdarzeniem z przeszłości - czas past perfect.

3. However

Uzasadnienie: however (jednakże) wprowadza kontrastową, odmienną od wcześniejszej informację.

4. on

Uzasadnienie: test sth on sb -  testować coś na kimś. 

5. working

Uzasadnienie: when he wasn't working, ... - kiedy nie pracował, ...


Zadanie 3

1. like

Uzasadnienie: what would it be like? - jakie by było?

2. number

Uzasadnienie: number of people - liczba ludzi.

3. which

Uzasadnienie: by which - za pomocą którego; zaimek względny which odnosi się do rzeczownika process. 

4. making

Uzasadnienie: make strides - robić postępy; tutaj w czasie present continuous (making).

5. be

Uzasadnienie: sytuacja, która (nie) będzie w trakcie wykonywania w konkretnym momencie w przyszłości - czas future continuous.
