Zadanie 2

W celu podkreślenia, że coś nie byłoby możliwe, gdyby nie ktoś albo coś innego, możemy użyć następujących konstrukcji:

  • if it weren't for/ but for, np.:

If it weren't for the rain/ But for the rain, we would go to the beach. (Gdyby nie deszcz, poszlibyśmy na plażę.)

  • if it hadn't been for/but for, np.:

If it hadn't been for my mum's support/ But for my mum's support, I wouldn't have succeeded. (Gdyby nie wsparcie mojej mamy, nie osiągnąłbym sukcesu.)

 

1. If it hadn't been for the alarm clock, I would have overslept.  

2. If it weren't for my friends, I'd have to deal with this mess alone.

3. If it weren't for the decorations, this place would be really ugly.

4. If it hadn't been for my parents' support, I wouldn't have become a video game designer. 


Zadanie 3

1. If it weren't for the pictures

Uzasadnienie: podkreślenie, że coś nie byłoby możliwe, gdyby nie coś innego w drugim trybie warunkowym - if it weren't for + noun phrase. 

2. unless you reveal

Uzasadnienie: sytuacja, która może się wydarzyć - first conditional; zamiast if (jeśli) - unless (jeśli nie)

3.  wouldn't go

Uzasadnienie: hipotetyczna sytuacja z teraźniejszości i jej potencjalne przyszłe następstwa - second conditional.

4. it weren't for my arts teacher

Uzasadnienie: podkreślenie, że coś nie byłoby możliwe, gdyby nie ktoś inny w drugim trybie warunkowym - if it weren't for + noun phrase. 

5. provided that he pays 

Uzasadnienie: sytuacja, która może się wydarzyć - first conditional; zamiast if (jeśli) - provided that (pod warunkiem, że).

6. it hadn't been for the money which

Uzasadnienie: podkreślenie, że coś nie byłoby możliwe, gdyby nie coś innego w trzecim trybie warunkowym - if it hadn't been for + noun phrase. 

7. If I were you 

Uzasadnienie: hipotetyczna sytuacja z teraźniejszości i jej potencjalne przyszłe następstwa - second conditional.


Zadanie 4

1. won't sell don't sell

Uzasadnienie: sytuacja, która może się wydarzyć w przyszłości - first conditional.

2. he asked he had asked

Uzasadnienie: hipotetyczna sytuacja z przeszłości i jej niedoszłe przeszłe następstwa - third conditional.

3. as long that as long as

Uzasadnienie: sytuacja, która może się wydarzyć w przyszłości - first conditional; zamiast if (jeśli) - as long as (o ile).

4. had called would call

Uzasadnienie: hipotetyczna sytuacja z teraźniejszości i jej potencjalne przyszłe następstwa - second conditional.

5. you'd have probably been you'd probably be

Uzasadnienie: hipotetyczna sytuacja z przeszłości i jej skutki w teraźniejszości - if + past perfect, would + bezokolicznik.

6. took had taken 

Uzasadnienie: hipotetyczna sytuacja z przeszłości i jej skutki w teraźniejszości - if + past perfect, would + bezokolicznik.
